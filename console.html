console.log를 찍어보기위한곳
 
숫자와 문자열 등의 타입들이 서로 다른 타입일 경우

==, !=는 동일한 타입으로 간주하지만, >자료형 변환을 한다

===, !==는 다른 타입으로 간주한다. > 자료형 변환을 하지않는다

console.log(1 == '1'); // true 같은 데이터 타입 간주
동등 연산자 (==)>>일치할때 true를 반환
두 피연산자의 자료형을 일치시킨 후, 엄격하게 비교를 수행


console.log(1 === '1'); // false 다른 데이터 타입 간주
일치 연산자 (===) >>일치할때 true를 반환
자료형 변환 없이 두 피연산자가 엄격히 같은지 판별



console.log(1 != '1'); // false 같은 데이터 타입 간주
부등 연산자 (!=) >>>불일치할때 true를 반환
두 피연산자의 자료형을 일치시킨 후, 엄격하게 비교를 수행


console.log(1 !== '1'); // true 다른 데이터 타입 간주
불일치 연산자 (!==) >>>불일치할때 true를 반환
자료형 변환 없이 두 피연산자가 엄격히 같은지 판별
두 피연산자가 같지 않거나, 같은 자료형이 아닐 때 true를 반환
